## Project Overview
The Shalom Dev Portfolio is a premium, high-performance portfolio website built with **Next.js 15**, **Framer Motion**, and **Tailwind CSS**. It features a modern, elite design with smooth animations and full 10-language internationalization support.

### Core Technologies
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Internationalization:** `next-intl`
- **Language:** TypeScript
